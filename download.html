ImageWarp features an advanced two-level demo mechanism.

The Limited Demo has the following restrictions:

  • 14-day trial period
  • Only the Simulation device is supported (no GigE, FireWire, BitFlow, VFW, WDM or TWAIN devices)
  • Only demo images can be opened from the hard drive
  • Images can be saved in the ImageWarp proprietary format (IWD) only
  • Images cannot be copied to or pasted from the Clipboard
  • A number of selected functions are disabled
  • Execution of scripts is interrupted every 60 seconds

Click to download the Limited ImageWarp Demo.

If you are serious about testing ImageWarp and you find the Limited Demo too restrictive, you can convert it to a fully functional version by registering it online. Select the Register command in the Help menu of ImageWarp and follow the instructions.

Please note that your computer is required to have a network adapter in order for ImageWarp to run. It is also recommended to uninstall any previous version of ImageWarp before installing the new one.

The Registered Demo offers the following:

  • 21-day trial period
  • Only GigE Vision, FireWire and BitFlow camera interfaces are supported for video capture
  • Images can be saved in the ImageWarp proprietary format (IWD) only
  • Images cannot be copied to or pasted from the Clipboard
  • Only a limited amount of measurement data can be recorded or exported.
